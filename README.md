# Offer Acceptance Prediction in Recruitment Processes

## Overview

This People Analytics project applies Machine Learning techniques to predict whether a candidate will accept a job offer during a recruitment process.

The objective is to help Talent Acquisition teams improve hiring efficiency, reduce offer rejection rates, and optimize recruitment decision-making through predictive analytics.

---

## Project Contents

- Exploratory Data Analysis (EDA)
- Data preprocessing
- Feature engineering
- Classification modeling
- Model evaluation
- HR and business insights

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## Dataset

The dataset contains simulated recruitment process information, including:

- Candidate experience
- Salary expectations
- Interview scores
- Offer conditions
- Recruitment stages
- Final offer acceptance outcome

---

## Business Objective

Predicting offer acceptance allows organizations to:

- Improve recruitment conversion rates
- Anticipate candidate behavior
- Optimize compensation strategies
- Reduce time-to-hire
- Increase hiring efficiency
